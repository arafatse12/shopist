Server Requirements

You will need the following major things to run the shopist.

  1. PHP >= 5.6.4
  2. OpenSSL PHP Extension
  3. PDO PHP Extension
  4. Mbstring PHP Extension
  5. Tokenizer PHP Extension
  6. PHP Fileinfo extension enable from php.ini

Be careful with your server requirements while installing shopist.

How to Install Shopist #back to top

  1. Unzip the shopist folder.
  2. Copy all files from shopist folder to your server
    3. Then go index.php file from root folder. 
                                    1. Find 
                                require __DIR__.'/../shopist/bootstrap/autoload.php';
                                and change with the
                                require __DIR__.'/../bootstrap/autoload.php';

                                2. Find
                                $app = require_once __DIR__.'/../shopist/bootstrap/app.php';
                                and change with the
                                $app = require_once __DIR__.'/../bootstrap/app.php';
  3. Open .env file from root folder and change DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD with your server DB information
  4. Then hit your domain. Suppose your domain name is "example.com". So this case hit on http://example.com.
  5. You will get admin registration form. Fill the information for the super admin access.

    6. If you have sub directory on your server, then change with this way

                                1. Find 
                            require __DIR__.'/../shopist/bootstrap/autoload.php';
                            and change with the
                            require __DIR__.'/../sub directory name/bootstrap/autoload.php';

                            2. Find
                            $app = require_once __DIR__.'/../shopist/bootstrap/app.php';
                            and change with the
                            $app = require_once __DIR__.'/../sub directory name/bootstrap/app.php';

Your admin url
https://example.com/admin/login
if you have sub directory
https://example.com/sub directory name/admin/login

Your frontend url
https://example.com
if you have sub directory
https://example.com/sub directory name

Cache clear url
https://example.com/clear-cache
if you have sub directory
https://example.com/sub directory name/clear-cache

Upsells and Cross Sells Products Setup #back to top

Up-sells for a product are items that you would like your customer to buy instead of the product that he is viewing, and they are pricey, better quality, etc.

Cross-sell items can appear both in the product page and in the shopping cart but they are a bit like an impulse buy – similar to items at the cash registers in grocery stores.

reCAPTCHA is a CAPTCHA-like system designed to establish that a computer user is human (normally in order to protect websites from bots) and, at the same time, assist in the digitization of books.
